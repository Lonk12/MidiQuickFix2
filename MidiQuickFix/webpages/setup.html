MidiQuickFix is available both as a pre-built executable jar file or source code from the downloads page and also as source code from CVS (for those who want the latest bugs fixes)

Prerequisites
A properly installed j2re version 1.4 or greater is all that is needed.
I use NetBeans 4.0 as my development environment so, if you would like to work on the code, it would probably be worth installing that too.

Jar file
Download the jar file.
Run the program with
java -jar MidiQuickFix.jar
